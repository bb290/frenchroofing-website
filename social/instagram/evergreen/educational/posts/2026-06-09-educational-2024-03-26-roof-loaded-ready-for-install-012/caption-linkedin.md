# Caption (linkedin)

A loaded roof ready for install is one of those job-site moments that tells you whether the prep was done right.

Before a single shingle goes down, the underlayment needs to be smooth and properly lapped, ice and water shield needs to be set at the eaves, valleys, and any penetration points, and the bundles need to be staged so the crew isn't hauling material mid-row. When all of that is in order, the install goes cleanly. When it isn't, you're chasing problems after the fact.

For property managers and GCs, this is the stuff worth asking about before you hire a roofing sub. Not just "how long will it take" but "what does your prep sequence look like." A crew that can answer that clearly has done it enough times to have a system.
