# French Roofing — Quick Reference Card

Pin this to a Notion doc, Slack canvas, or a wall. Glance at it during reviews.

## Colors

| Token | Hex | Where it's used |
|---|---|---|
| Primary Yellow | `#ffbd59` | Primary CTAs, highlights, accents |
| Secondary Orange | `#d85024` | Secondary CTAs, hovers, emphasis, focus rings |
| Tertiary Blue | `#092e5e` | Headlines, deep sections, button text on yellow |
| Neutral Grey | `#3e3d3b` | Body text |
| Background | `#fdfcfa` | Default page background (warm off-white) |
| Surface (Cream) | `#f5f1ea` | Cards, raised sections |
| Border | `#e5e0d6` | Card borders, input borders, dividers |
| Text Secondary | `#6b6a67` | Subheads, secondary copy |

| Success | `#4a7c4e` | Confirmations |

## Components (one-liners)

| Component | Default |
|---|---|
| Primary button | `#ffbd59` fill · `#092e5e` text · 8px radius · no shadow |
| Secondary button | Outlined `#092e5e` · transparent fill · 8px radius |
| Card | `#f5f1ea` on `#fdfcfa` · 12px radius · 1px `#e5e0d6` border · no shadow · 24–32px padding |
| Input | White fill · 1px `#e5e0d6` border · 8px radius · `#d85024` focus border |
| Table | No zebra · 1px bottom borders only · 56px+ row height |
| Image | Real crews + PNW homes · natural light · 4–8px radius · no filters |

## Voice — words

| ✅ Use | ❌ Avoid |
|---|---|
| we, your roof | premier, industry-leading |
| neighbor, the crew | rockstar, synergy |
| free assessment | amazing team |
| Got You Covered! (1 in 6 posts max) | high-pressure CTAs |

## Aesthetic in one sentence

Sean in Birkenstocks explaining your roof over the fence — laid back, photo-forward, generous whitespace, never corporate.
